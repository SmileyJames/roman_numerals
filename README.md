# Roman Numerals

A Python program/function to generate roman numerals from integers.

## Usage

```
python3 roman.py 24
```

usage: roman.py [-h] N

Generate roman numerals.

positional arguments:
  N           An integer to convert to roman numerals.

optional arguments:
  -h, --help  show this help message and exit
